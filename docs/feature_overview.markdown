# Overview of Features

## Supported Mesh Types
* OpenFOAM PolyMesh
* Fluent case (.cas)
* Fluent mesh (.msh)
* Gmsh (.msh)
* I-deas Univesl (.unv)


## Mesh manipulation
* Scale
* Translate
* Rotate


## Turbulent Models
* Laminar
* Spalart-Allmaras
* k-epsilon
    * Standard
    * RNG
    * Realizable
* k-omega
    * SST


## Cell Zone Configuration
* Zone Types
    * Multiple Reference Frame (MRF)
    * Porous Zone
    * Sliding Mesh
    * Actuator Disk
* Source Terms
* Fixed Values


## Monitoring values
* Residual Values
* Force Coefficients
* Point Values
* Surface Values
* Volume Values


## Parallel Processing
* MPI Parallel Processing
